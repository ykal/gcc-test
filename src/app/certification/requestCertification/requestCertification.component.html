Assessment One Tasks


  • 1. Create your own story line and write a scratch program forming an animation for the story line. The animation should include 1, the conversation between characters in the story 2, at least two or more background changes 3, Movement of characters; transition of characters from one costume to another.

  • 2. Create your own style maze game. The program should have a maze which has different levels and once a player has solved one level it will directly transition to the next level of the game. Thus the game is expected to have at least two levels or more.

  • 3. Write a program that takes the sides of the polygon as an input and draw the polygon with the side given by the input.

  • 4. Create a pong game. The pong game has one ball and two paddles one for each player. The two players should have their own unique keys to control the movement of the paddle up and down. The environment of the game is shown in the figure. The point of the game is preventing the ball which moves randomly from one point to other from touching the red line. The game should have a variable to store the scores of players and the duration of the game should be limited to a certain period of time.
